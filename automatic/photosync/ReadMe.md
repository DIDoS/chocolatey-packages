## Features

- Unlimited transfers
Transfer photos & videos between iPhone, iPad, Android, Windows PC, Mac, Linux, NAS, cloud services & more…

- Cross-platform
Number one cross-platform solution with native apps for iOS (iPhone, iPad), Android, Windows PC and Mac. Support for Linux too!

- Retain full control
Total user control and fully customizable at all times. Be totally independent of cloud services!

- Reliable & secure
Operating 8+ years in the market and continuously upgraded. An indispensible tool for photo and video transfers!
